## 👋 你好！

感谢你发现了 Producktivity！

这是一个轻量级的 Windows 桌面宠物应用。一只像素风小鸭子会在你的屏幕底部来回走动，定时给你发送鼓励消息和表情包，帮助你保持专注、按时休息。如果你忍不住打开了不该打开的应用，它还会帮你关掉，然后嘎嘎叫着教训你两句。

我们真诚地希望这只小鸭子能成为你工作和学习时的好伙伴。

---

## ✨ 功能一览

### 🦆 桌面宠物鸭
- 像素风鸭子在屏幕底部自由行走，带有走路动画
- 可以用鼠标左键拖拽鸭子移动位置
- 右键点击鸭子会收到一条随机回复
- 鸭子头顶显示自定义名牌

### 🪹 鸭子窝
- 可拖拽的鸭子窝，右键打开功能菜单
- 让鸭子"回窝休息"或"出巢活动"
- 打开聊天框、设置面板
- 拖动鸭子窝时鸭子会跟随移动

### 💬 聊天框
- 美观的聊天界面，鸭子会在这里发送所有消息
- 支持文字消息和表情包图片消息
- 可滚动查看历史消息，显示消息时间戳
- 可拖拽移动、可调整高度

### 📨 随机消息系统
- 鸭子按照设定的时间间隔，随机发送鼓励、提醒和搞怪消息
- 内置 **51 条精心编写的消息**，分为三种风格：
  - 🎉 **活泼搞怪** — 嘎嘎出没，欢乐提醒
  - 💪 **鼓励打气** — 轻松有劲，温暖加油
  - 🌿 **温柔沉稳** — 像朋友在身边，安静陪伴
- 也会随机发送已启用的表情包
- 文字与表情包的发送比例可自由调节

### 🖼️ 表情包管理
- 导入 PNG / JPG / GIF 格式的表情包
- 为每个表情包设置 0～3 星评级（星级越高，被发送的概率越大）
- 单击顶部色条切换启用/禁用
- 双击缩略图预览大图

### ⚙️ 设置中心

| Tab | 功能 |
|-----|------|
| **鸭子名称** | 自定义鸭子的名字，最多 20 个字符 |
| **表情包管理** | 导入、预览、评级、启用/禁用表情包 |
| **控制** | 调整消息间隔（LB/HB）、文字比例（K）、管理拦截应用列表 |
| **数据** | 表情包上传数、文字消息数、表情包消息数、拦截次数、运行总时长 |

### 🚫 应用拦截
- 设置需要拦截的应用列表
- 程序每 3 秒扫描正在运行的进程
- 检测到黑名单应用时自动关闭，并发送趣味劝阻消息
- 内置 **34 条风格各异的拦截提示**（搞怪 / 正式 / 温柔）
- 记录累计拦截次数

### 📊 数据统计
- 上传表情包总数
- 文字消息发送总数
- 表情包消息发送总数
- 应用拦截总次数
- 程序累计运行时长
- 数据自动保存，跨会话累计

## 🎮 操作指南

| 操作 | 效果 |
|------|------|
| 左键拖拽鸭子 | 移动鸭子位置 |
| 右键点击鸭子 | 收到一条随机消息 |
| 右键点击鸭子窝 | 打开功能菜单 |
| 拖拽鸭子窝 | 鸭子窝和鸭子一起移动 |
| 聊天框顶部拖拽 | 移动聊天窗口 |
| 聊天框底部拖拽 | 调整聊天窗口高度 |
| 设置 → 控制 → 保存 | 参数立即生效，无需重启 |
